# d'LITTIcious Restaurant Website

## Local development

```sh
# Install dependencies
npm install

# Start dev server
npm run dev
```

## Build

```sh
npm run build
```

## What technologies are used for this project?

This project is built with:

- Bun 1.3.4
- Vite 7.2.7
- TypeScript
- React 19.2.1
- shadcn-ui
- Tailwind CSS
